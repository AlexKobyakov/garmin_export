## 🔧 Техническая реализация

### Основные функции
1. **Выбор слоёв**: автоматическое сканирование проекта
2. **Настройки экспорта**: Family ID, Map ID, пути к файлам
3. **Сопоставление стилей**: JSON-редактор с валидацией
4. **Уровни карты**: настройка детализации Level0-3
5. **Компиляция**: многопоточная обработка с прогрессом

### Процесс экспорта
1. Подготовка и валидация данных
2. Обработка векторных слоёв
3. Генерация MP файла (Polish format)
4. Компиляция через mkgmap в IMG
5. Очистка временных файлов

### Обработка ошибок
- Валидация входных данных
- Проверка зависимостей (Java, mkgmap)
- Обработка ошибок геометрии
- Логирование всех этапов процесса
